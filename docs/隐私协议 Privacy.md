# 隐私协议 Privacy

上次更新 2026-03-16

## 第一部分 收集你的信息

在访问网站的时候，你需要 **启用** 浏览器的 `JavaScript` 功能来准许 **信息的正常获取** 与 **功能使用**

### 怎么收集信息？

因此，在访问时，这些（本站内置或第三方）功能会通过这些途径获取你的信息：

- Umami
- Cloudflare
- GitHub
- Vercel/Netlify
- Aliyun
- Microsoft Clarity

### 那会收集到什么信息？

包括但不限于：

- 浏览器 User-Agent，如 "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:149.0) Gecko/20100101 Firefox/149.0"
- IP 地址与基于 IP 定位的地理位置，如 61.170.\*.\* 上海市
- 设备系统信息（基于 UA），如从前 UA 提取出来的 Windows NT 10.0 x64
- 浏览方式（基于第三方服务而异），例如 Clarity 会收集操作方式（触控 / 桌面）、浏览时长、浏览轨迹、网络情况等

## 第二部分 使用你的存储

为加快网站访问，因此会将部分资源文件的 Keep-Alive 时间调整的相对较长

若有需要，你可以自行选择清除

## 第三部分 如何与其他人共享、转让你的个人信息

本人 **坚决抵制** 将隐私信息肆意共享给其他公司、组织与个人的行为，
同时 **坚决不会** 将个人信息转让给其他公司、组织与个人。

但请注意，如果遭遇不可避免的意外（例如依法查询）
此时本人将有权将你的信息 **依法** 共享或公开。
