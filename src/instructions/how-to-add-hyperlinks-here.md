## 添加友链

**大佬，页面，加加**

怎么在这里添加友链呢？

Step 1:
确保您的网站有 **可公开访问** 的 **首页** 和 **友链** 页面，
确保您的网站是可以 **正常被访问到** 的，
确保我的网站是可以 **正常被你访问到** 的。

Step 2:
在您的 **友链** 页面根据下方的信息来添加本站友链。

Step 3:
通知我，在一定时间内我会把友链添加到这里。

友链信息表：

```yaml
CILXRY:
  site-name: CILXRY 纪事小栈
  site-link: https://blog.cilxry.cc
  site-avatar: https://avatar.cilxry.cc
  description: 许愿 如所愿 皆所愿
```

若您已理解以上内容，烦请填写在下方commit：

[前往 GitHub 提友链Issue](https://github.com/CILXRY/cilxry-corner/issues)
